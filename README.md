# SQL-Python

Este proyecto contiene una colección de notebooks donde se exploran conceptos de SQL aplicados con Python, usando herramientas como `sqlite3` y `pandas`.

## 📘 Contenido

Organizado en capítulos:

- **Capítulo 1**: Fundamentos de SQL en Python.
- **Capítulo 2**: Consultas intermedias y manejo de datos.
- **Capítulo 3**: Aplicaciones y análisis de datos.

Cada capítulo contiene ejemplos prácticos escritos en Jupyter Notebook.
